# Wellcome 文档生成器 — 操作指南

> 访问地址：`http://你的IP:8501`（向 Vincy 索取）

---

## 一、注册和登录

1. 打开网页 → 点「注册」标签
2. 填写邮箱、用户名、密码 → 提交
3. 等待管理员审核通过后即可登录使用

> ⚠️ 注册后需等待管理员审核，审核通过前无法使用任何功能。

---

## 二、同事（业务端）日常操作

### 生成确认函 + Invoice

1. 登录后默认在「📄 生成文档」页面
2. 填写以下信息：

| 字段 | 说明 |
|------|------|
| 客户简称 | 下拉选择已录入的客户 |
| 申请日期 | 默认今天 |
| 项目编号 | 如 WELL20260801001 |
| 项目名称 | 如 XXX品牌7月UGC 150篇 |
| 客户品牌名 | 品牌的社交名称 |
| 币种 | USD 或 RMB |
| 项目金额 | 合同金额 |
| 执行地点 | 城市 |
| 执行周期 | 如 2026年7月-8月 |
| 预计拍摄时间 | 如 2026年7月 |
| 总发布篇数 | 如 150 PHOTO POSTS |
| 到期日 | Invoice 付款截止日期 |
| 合作内容 | 默认「UGC铺量」 |
| 发布平台 | 默认「小红书」 |
| 预估成本金额 + 币种 | 项目预估成本 |
| 成本构成 | 如 KOL费用2000, 拍摄500... |

3. 勾选「生成后提交财务审核」→ 点「🚀 生成文档」
4. 下载确认函和 Invoice
5. 复制邮件文案发送给客户

### 客户管理

- 侧边栏 → 「👥 客户管理」
- 可新增、编辑、删除客户信息
- 新增客户时，简称和公司全称为必填

### 查看历史

- 侧边栏 → 「📋 项目历史」
- 查看所有项目及审批状态
- 已通过的项目可直接下载盖章 PDF

---

## 三、财务操作

### 审核 Invoice

1. 登录后侧边栏 → 「💰 财务审核」
2. 看到待审核列表，点击「📥 下载 Invoice」核对内容
3. 核对无误 → 点击「✅ 通过」
4. 系统自动生成盖章 PDF，页面顶部弹出下载按钮
5. 点击下载盖章 PDF，发给客户

> 如有问题，点「❌ 驳回」

### 开具收据

1. 侧边栏 → 「🧾 开具收据」
2. 选择「从已通过项目生成」→ 下拉选择对应项目
3. 系统自动填入客户和项目信息，财务只需填写：
   - **实收金额**（可能与合同金额不同，如分批收款）
   - **付款日期 / 到款日期**
   - **付款方式**（银行/现金/转账）
4. 点击「生成收据并盖章」
5. 下载盖章收据 PDF，发给客户

> 也可选「手动填写」模式，用于非系统内项目的收据。

## 四、常见问题

**Q：刷新页面要重新登录？**
不会。系统会记住登录状态 24 小时，关闭浏览器再打开也不用重新输入密码。

**Q：找不到审批通过的盖章 PDF？**
1) 财务审核页面通过后直接弹下载
2) 「📋 项目历史」中每个已通过项目都有下载按钮

**Q：需要新增客户？**
「👥 客户管理」→ 新增客户，填写简称和全称即可。之后生成文档时下拉自动出现。

**Q：谁来审核我的注册？**
管理员（Vincy）在「🔒 用户审核」页面审批新用户注册，并可设置用户角色。

---

## 五、角色说明

| 角色 | 权限 |
|------|------|
| 管理员 | 所有功能 + 用户审核 + 角色管理 |
| 财务 | 生成文档 + 财务审核 + 开具收据 |
| 普通用户 | 生成文档 + 客户管理 + 查看历史 |

> 新注册用户默认是「普通用户」，需要管理员在用户审核页面设为「财务」角色才能审核和开收据。
