# Wellcome 财务自动化平台 — 交接指南

## 你需要移交的三个账号

| 平台 | 账号 | 怎么移交 |
|------|------|------|
| **GitHub** | vincyleeeeeee | Settings → Account → Transfer ownership 给接手人 |
| **Streamlit Cloud** | 用 GitHub 登录 | 接手人用自己的 GitHub 登录后，把仓库 fork 过去重新部署 |
| **Supabase** | 你的邮箱 | Project Settings → Transfer project 给接手人 |

---

## 接手人操作步骤

### 1. 接手 Supabase 数据库
- 数据库里存了所有用户、客户、项目记录
- 去 Supabase → Project Settings → **Transfer Project** → 输入接手人邮箱
- 接手人接受后，数据库归他管理

### 2. 接手 GitHub 代码
- 去 GitHub → Settings → **Transfer ownership**
- 或者：接手人 fork 仓库到自己的账号
- 仓库地址：`github.com/vincyleeeeeee/wellcome-finance-app`

### 3. 重新部署到 Streamlit Cloud
- 接手人打开 https://share.streamlit.io
- 用自己 GitHub 登录
- 点 New App → 选接手人账号下的 wellcome-finance-app
- Main file path: `app.py`
- Deploy

### 4. 配置 Secrets
- Streamlit Cloud → App Settings → Secrets
- 填入 Supabase 的 URL 和 Key
- 格式：
```
SUPABASE_URL = "https://xxxxx.supabase.co"
SUPABASE_KEY = "eyJhbGci..."
```

### 5. Supabase SQL Editor 建表
- 接手人进入 Supabase → SQL Editor
- 运行以下 SQL 建表（或从项目里找到建表语句）
- 表包括：users, clients, projects

## 运行方式

1. 本地运行：`cd invoice-app && python3 -m streamlit run app.py`
2. 云端运行：Streamlit Cloud 自动部署
3. 访问地址：接手人部署后的 Streamlit Cloud URL
